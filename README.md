# 项目简介

## 核心功能
- 员工上下班打卡
- 考勤数据统计分析
- 部门管理
- 员工信息维护
- 基于JWT的身份验证机制
- 细粒度的权限控制

## 适用场景
- 企业内部考勤管理
- 人员出勤情况监控
- 部门和员工考勤数据统计
- 高效精准的人员管理

## 技术特点
- 采用前后端分离架构
- 使用RESTful API实现数据交互
- 基于Python和Django开发
- 数据存储于MySQL数据库
- 使用vue3进行前端开发
